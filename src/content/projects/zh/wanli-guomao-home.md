万里国贸之家是面向浙江万里学院国际经济与贸易专业的学习辅助网站。我用 AI 工具（TRAE、Codex）独立完成从设计、开发到上线的全过程，把同专业同学常用的信息收进一个清晰入口。

## 我的工作

- 基于 HTML / CSS / JavaScript 搭建多页面响应式网站，适配桌面、平板、手机三类设备，实现移动端导航、平滑滚动、二维码弹窗等交互。
- 围绕国贸专业整合专业介绍、课程体系、学科竞赛、考证指南、就业方向五类资源，做成一站式信息入口。
- 通过 GitHub 托管 + GitHub Pages 部署，绑定自定义域名 www.wlgmzj.cn 正式上线，并经 40+ 次提交持续迭代。

## 项目价值

这个项目是我把国际经济与贸易专业背景和前端开发结合起来的一次实践：不堆砌炫技页面，而是从身边真实的学习场景出发，快速做出可以公开访问、能持续维护的产品。
